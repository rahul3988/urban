# Jeb Dekho - Admin Panel

React + TypeScript web application for administrators to manage the entire platform.

## Features

- Admin authentication
- Analytics dashboard
- User management (customers, vendors, delivery partners)
- Service management
- Platform configuration

## Screens

1. **Authentication**
   - Login

2. **Main Panel**
   - Analytics Dashboard
   - User Management

## Setup

```bash
npm install
npm start
```

## Tech Stack

- React 18
- TypeScript
- React Router v6
- Tailwind CSS
